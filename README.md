# ZeroJudge Solutions

這個倉庫整理了我在 [ZeroJudge](https://zerojudge.tw/) 練習的題目解答。檔案依照題號開頭的英文字母分類，檔名即為題號（例如：`A/a001.py` 對應題目 `a001`）。

## 📚 目錄結構

```
├── A/  # 以 A 開頭的題目
├── B/  # 以 B 開頭的題目
├── C/  # 以 C 開頭的題目
├── D/  # 以 D 開頭的題目
├── E/  # 以 E 開頭的題目
├── F/  # 以 F 開頭的題目
├── G/  # 以 G 開頭的題目
├── H/  # 以 H 開頭的題目
├── I/  # 以 I 開頭的題目
├── O/  # 以 O 開頭的題目
├── Q/  # 以 Q 開頭的題目
└── R/  # 以 R 開頭的題目
```

- 以題號第一個字母分資料夾
- 檔名即題號，副檔名為解題語言（多數為 `.py`，少數為 `.c` 或 `.cpp`）

## ⚠️ 注意事項

- 解答以 ZeroJudge 題目輸入輸出格式為準，請透過標準輸入提供測資
- 內容主要為個人解題紀錄，並非題目最佳解或唯一解
- 建議先自行嘗試解題，再參考解答以獲得最佳學習效果
